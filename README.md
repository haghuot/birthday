# 生日祝福小网站（可分享链接）

这是一个**纯静态**的小页面：对方打开链接即可查看；支持用链接参数自定义“标题 / 对方名字 / 署名 / 正文”，并带有一个简单的彩带烟花动效。

## 文件说明

- `index.html`：主页面
- `style.css`：样式
- `script.js`：功能（读取链接参数、复制链接、烟花、主题切换）

## 本地预览

直接双击打开 `index.html` 即可预览（也可以用任意静态服务器打开）。

> 注意：要让“分享链接”真的能被对方在手机上打开，建议把它发布到静态托管平台（见下文）。

## 一分钟发布成可分享链接（推荐：Netlify）

1. 打开 Netlify 的拖拽发布页：`https://app.netlify.com/drop`
2. 把整个 `生日祝福` 文件夹拖进去
3. 等待几秒，会得到一个 `https://xxxx.netlify.app` 的链接
4. 把这个链接发给对方即可

## 发布成可分享链接（GitHub Pages）

1. 在 GitHub 新建一个仓库（比如叫 `birthday`）
2. 把这 3 个文件上传到仓库根目录：`index.html`、`style.css`、`script.js`
3. 进入仓库 **Settings → Pages**
4. Source 选择 `Deploy from a branch`，Branch 选择 `main`，目录选 `/ (root)`，保存
5. 等一会儿就会生成一个 Pages 链接，发给对方即可

## 自定义祝福内容（通过链接参数）

支持参数：

- `to`：对方名字（可选）
- `from`：署名（可选）
- `title`：标题（可选）
- `msg`：正文（可选，支持用 `\n` 换行）
- `date`：页脚日期（可选）

示例（把域名换成你的实际链接）：

```
https://你的链接/index.html?to=小明&from=老朋友&title=生日快乐&msg=愿你眼里有光，心里有爱。\n新的一岁，继续闪闪发光！
```

小提示：

- 中文和特殊字符会被自动编码（页面里会展示一条“示例链接”，也可以直接复制参考）
- 为了安全，`msg` 会按纯文本显示（不会解析 HTML）
